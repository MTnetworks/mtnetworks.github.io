# 知识库管理系统

一个现代化的知识库管理系统，提供丰富的文档编辑和管理功能。支持多种文档格式，包括富文本、Markdown、块式编辑器和代码编辑器，帮助用户高效管理个人知识资产。

## 功能特点

### 用户界面
- 现代化的响应式布局系统
- 可折叠的侧边栏导航
- 顶部导航栏（面包屑、用户信息、快捷操作）
- 仪表板页面（统计概览、快速操作、最近文档）
- 文档列表管理页面（搜索、标签、筛选、批量操作）

### 编辑器功能
- 富文本编辑器：集成Quill.js，支持格式化、工具栏、自动保存
- Markdown编辑器：实时预览、语法高亮、分屏模式
- 块式编辑器：类似Notion的拖拽块编辑体验
- 代码编辑器：Monaco Editor集成，多语言支持、主题切换

### 文档管理
- 完整的CRUD操作（创建、读取、更新、删除）
- 文档搜索和筛选功能
- 批量操作（删除、导出）
- 文档复制和导出功能

## 技术栈

### 后端
- Node.js + Express.js
- MongoDB + Mongoose
- JWT身份验证
- 文件上传处理

### 前端
- React 18
- TypeScript
- Tailwind CSS
- React Router
- Axios
- 多种编辑器集成

## 快速开始

### 安装依赖

```bash
# 安装后端依赖
cd backend
npm install

# 安装前端依赖
cd ../frontend
npm install
```

### 启动开发服务器

```bash
# 启动后端服务器 (端口: 5000)
cd backend
npm run dev

# 启动前端开发服务器 (端口: 3000)
cd frontend
npm start
```

### 访问应用

打开浏览器访问 `http://localhost:3000` 即可使用知识库管理系统。

## 开发说明

- 后端API服务运行在 `http://localhost:5000`
- 前端开发服务器运行在 `http://localhost:3000`
- MongoDB数据库连接配置在后端环境变量中
- 支持热重载开发模式
